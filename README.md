# My Life in Weeks

A beautiful, fully editable, offline-first "life in weeks" visualization.  
One single HTML file. No backend. No tracking. Forever yours.

## Features
- See your entire life (up to 90 years) as 4,680 tiny dots
- Every change is auto-saved in your browser (`localStorage`)
- Add unlimited memories with photos, emojis, custom colors
- Click any memory → full-screen photo + confetti
- Multiple themes (Dark, Barbie, Matrix…)
- Export as 8K poster
- Backup/restore your entire life as a tiny `.json` file
- Works offline forever – even in 2073

## How to use
1. Open `index.html` in any browser
2. Click **"Edit My Life"**
3. Change your birthday & add memories
4. Everything saves automatically
5. Share the single file with anyone – their edits stay on their device
